# Smart Budget Planner

## Description
The Smart Budget Planner is a simple Java console application that helps users manage a monthly income to record expenses. Users can input their monthly income, add expense entries (name and amount), view total spending, and see the remaining balance. This project focuses on Java fundamentals.

## Feature
- Set monthly income
- Add expense entries (name + amount)
- View list of expenses
- Show total expenses and remaining balance
- Exit the program

## Future Improvements
- Save/load data from a text file
- Edit or remove existing expenses
- Add categories and monthly reports
- GUI (JavaFX or Swing) or persistent database
